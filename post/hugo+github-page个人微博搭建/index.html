hugo+github page个人微博搭建

Feb 18, 2025 One minute  

hugo安装

1.安装hugo

从官网下载最新hugo最新版本
window使用Chocolatey安装hugo扩展版本：

choco install hugo-extended

配置环境变量 其他必备安装：

  • 安装git
  • 安装golang

hugo新建站点

在目录下，通过命令新建：

hugo new site sitename

hugo主题安装

hugo官网提供了主题下载，hugo主题下载
我这里选择的是heyo主题，进到官网heyo主题详情页，根据提示下载主题到本地：

cd hugo站点项目根目录
git clone https://github.com/LucasVadilho/heyo-hugo-theme heyo
cd heyo/exampleSite && hugo server --themesDir ../..

部署到github

在github中可以通过pages来设置静态网页托管，具体设置如下：


重要！将config.toml文件中的baseUrl值修改为yourname.github.io地址
通过hugo命令来生成静态页面，这里可以在自定义文件夹内新建content和static目录用来存放博客文件和静态文件，可以通过 –themesDir path 参数来制定主体目录

cd heyo/exampleSite
hugo --themesDir ../..

此时，当前目录下会生成public文件夹，cd到public目录下，通过git上传文件：

cd public
git init    #初始化仓库
git remote add origin https://github.com/101bingo/101bingo.github.io.git  #链接远程仓库
git branch --set-upstream-to=origin/master master
git add .
git commit -m "first commit"
git push -u origin master

后续发布博客，先执行hugo –themesDir ../..，再到public目录下提交代码：

git add .
git commit -m "commit info"
git push
